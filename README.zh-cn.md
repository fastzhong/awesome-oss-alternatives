## 选取标准

如果符合以下条件，开源公司将被添加到列表中：

1. 其产品主要基于开源软件
2. 它有一个解决了类似业务问题的知名闭源竞争对手
3. 这是一家过去 10 年内成立的，以营利为目的的私营公司
4. 它在 GitHub 上的仓库有 100 个以上的星

创业世界里的事情变化非常快，所以这个列表既不能保证完整，也不能保证 100% 最新。
